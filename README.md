# RaspisanieSPO — тарификация и расписание для СПО

Десктоп-приложение (Python + PySide6) для завуча/зам. директора по УР: справочники преподавателей и групп, производственный календарь, учебный план, распределение часов по преподавателям (тарификация), автозаполнение расписания и отчёты с экспортом в Excel/PDF.

## Установка и запуск

### Готовые сборки

Скачайте последний релиз со страницы **Releases** репозитория:
- **macOS** — `RaspisanieSPO-macOS.zip`, распакуйте и перенесите `RaspisanieSPO.app` в `Программы`. При первом запуске macOS может попросить разрешить запуск неподписанного приложения (правый клик → «Открыть»).
- **Windows** — `RaspisanieSPO-Windows.exe`, запускайте напрямую. Windows SmartScreen может предупредить о неизвестном издателе (сборка не подписана платным сертификатом) — выберите «Подробнее» → «Всё равно выполнить».

### Запуск из исходников (для разработки)

```bash
git clone <ссылка на репозиторий>
cd raspisanie-spo
python3 -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Пароль

При первом запуске стоит пароль по умолчанию: **kolinet**. Сменить его можно в **Настройки → Пароль** (нужно ввести текущий и новый дважды). Пароль хранится в базе не в открытом виде, а как соль + PBKDF2-хеш.

## Структура приложения

Четыре основные вкладки: **Тарификация**, **Расписание**, **Отчёты**, **Настройки**.

### Настройки

| Вкладка | Что делает |
|---|---|
| Преподаватели | ФИО, кабинет, список предметов (предметы можно добавлять прямо из формы) |
| Программы | Специальности: код, название, срок обучения |
| Группы | Название группы, привязка к программе (специальности), курс |
| Календарь | Учебный год, недели по I и II полугодию с количеством часов каждая (быстрое добавление пачкой недель одинаковой длины); внизу — автосумма часов за полугодие |
| Учебный план | Для выбранной программы+курса — список предметов с часами теории/практики/экзамена и типом занятия (обычное/учебная практика/лабораторно-практическое — влияет на автозаполнение расписания) |
| Звонки | Время начала и окончания каждой пары — подставляется в расписание при экспорте |
| Пароль | Смена пароля входа в приложение |
| О программе | Текущая версия, кнопка проверки обновлений на GitHub |

### Тарификация

1. **Назначения** — выбираете группу, добавляете преподавателя на предмет из учебного плана этой группы (по программе и курсу). Показывается план по часам, есть перенос части часов между полугодиями (если преподаватель не помещается в лимит одного полугодия).
2. **Сводка по преподавателям** — общая таблица часов по всем преподавателям за оба полугодия, с подсветкой превышения лимита, который считается по календарю выбранного учебного года.

### Расписание

Выбираете год → полугодие → неделю (список недель и лимит часов берутся из Календаря) → группу. Сетка 6 дней × 5 пар:

- **Двойной клик по ячейке** — вручную назначить/изменить/убрать пару (предмет+преподаватель берутся из уже сделанных назначений на вкладке «Тарификация»; можно указать замену другим преподавателем).
- **Автозаполнить расписание на полугодие** — заполняет пустые пары автоматически по назначениям тарификации, не трогая то, что уже стоит. Учитывает:
  - недельный лимит часов из календаря;
  - учебную практику — всегда блоком 6 часов (3 пары) в один день;
  - лабораторно-практические занятия (информатика и т.п.) — все часы предмета за неделю ставятся в один день;
  - обычные предметы — старается не повторять один и тот же предмет дважды за день;
  - занятость преподавателя и кабинета — не ставит конфликтующие пары в разных группах одновременно.
- **Пересобрать полугодие заново** — очищает все пары группы за полугодие (включая расставленные вручную) и строит заново по текущим назначениям.
- **Пересобрать только эту неделю** — то же самое, но только для выбранной недели.
- **Экспорт в Excel / PDF** — выгружает текущую неделю с временем звонков, лимитом часов и подсветкой превышения.

### Отчёты

Выбираете преподавателя — видите план по тарификации против фактически прочитанных часов (из расставленного расписания) по полугодиям, понедельную разбивку, и отдельно — кто заменял этого преподавателя и кого заменял он сам. Экспорт в Excel и PDF (с местом для подписи).

## Обновления

**Настройки → О программе → Проверить обновления** запрашивает последний релиз на GitHub и сравнивает версию. Если есть более новая — кнопка «Скачать новую версию» открывает страницу релиза в браузере (автоматическая замена файла приложения не выполняется — скачайте и переустановите сборку вручную).

## Сборка установщиков самостоятельно

```bash
pip install pyinstaller
pyinstaller raspisanie.spec --noconfirm
```

Результат — `dist/RaspisanieSPO.app` (macOS) или `dist/RaspisanieSPO.exe` (Windows). Иконки лежат в `assets/` (`icon.icns`, `icon.ico`).

В репозитории настроен GitHub Actions (`.github/workflows/build.yml`): при пуше тега вида `vX.Y.Z` автоматически собираются обе сборки и прикрепляются к релизу с этим тегом.

## Данные

База — SQLite. При обычном запуске из исходников файл лежит в `data/raspisanie.db` внутри проекта. В собранном приложении — в системной пользовательской папке (macOS: `~/Library/Application Support/RaspisanieSPO/`, Windows: `%APPDATA%\RaspisanieSPO\`), чтобы данные не терялись между запусками и не зависели от того, куда установлено приложение.
